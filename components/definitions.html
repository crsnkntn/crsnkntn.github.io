Definitions

Artificial Intelligence (AI)

A non-human system able to perform cognitive work in a set of domains.

Machine Learning (ML)

Using data to adjust mathematical models, with the goal of using those models as artificial intelligence.

Artificial General Intelligence (AGI)

A non-human system able to perform superior cognitive work in all domains.
